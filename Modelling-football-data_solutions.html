The data

The data we will be using for this activity can be found in the 2019_nfl_qb_data.csv file and contains Quarterback stats for all quarterbacks in the 2019 NFL season. It has been sourced from pro-football-reference.com and consists of the following variables:

  1. case_no: unique observation identification number
  2. player_name: quarterback player name
  3. player_id: unique player identification number
  4. team: team the quarterback plays for in the NFL
  5. age: age of player in years
  6. position: Uppercase QB indicates a primary starting quarterback, while lowercase qb indicates a secondary quarterback
  7. games_played: number of games played
  8. games_started: number of games started
  9. wins: number of games won when starting
  10. losses: number of games lost when starting
  11. draws: number of games drawn when starting
  12. completions: number of passes completed
  13. attempts: number of passes attempted
  14. cmp_pc: passes completed as a percentage of passes attempted
  15. yards: yards gained by passing
  16. touchdowns: number of passing touchdowns
  17. touchdown_pc: passing touchdowns as a percentage of passes attempted
  18. interceptions: number of interceptions
  19. interceptions_pc: interceptions as a percentage of passes attempted
  20. passer_rating: Passer rating (also known as quarterback rating). For a description of the Passer rating see here
  21. sacks: number of sacks
  22. yards_lost: number of yards lost due to sacks
  23. sack_pc: sacks as a percentage of passes attempted

Each row is one players total stats for the 2019 season.

Our question

Passer rating is a metric used to evaluate quarterbacks in the NFL, and it is calculated based on a quarterback’s completions, yards gained, touchdowns, interceptions and attempts.

When using metrics to evaluate players or teams, it is important to determine whether these metrics are valid measures of performance. In this case, we need to determine if passer rating is a good measure of success. So, our question becomes:

How does passer rating relate to success?

We can use the number of wins (wins) a team has when a particular quarterback starts as a measure of success. And then the questions we are trying to answer now are:

  • Do quartbacks with a higher passer rating have more wins?
  • Do quarterbacks with a lower passer rating have fewer wins?
  • How many wins can we expect to get when we have a starting quarterback with a particular passer rating?

We can use simple linear regression to answer these questions.

Step 4. Data transformation.

If you take a look at the data, not all Quarterbacks have started the same number of games (games_started), and therefore we need to normalise the wins variable per games started (i.e. we need to create a variable “win percentage”. We should also set a limit for the minimum number of games a quarterback has started in order to include them in the analysis.

For this step, do the following in one sequence of code:

  1. filter the data so only quarterbacks that have started a minimum of 10 games are included,
  2. create a new variable called win_percentage, that expresses the number of wins as a percentage of games_started

Save the result in a new object.

new_df <- df %>%
  filter(games_started >= 5) %>%
  mutate(win_percentage = wins / games_started * 100)

Conduct a linear regression to determine whether a quarterbacks passer rating influences win percentage. Interpret what the intercept and slope coefficients mean in “real world” terms.
